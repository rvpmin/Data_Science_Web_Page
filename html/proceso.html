Proceso de la Ciencia de Datos


Una vez definido el problema a resolver, se inicia el proceso de la Ciencia de Datos que consta de varias etapas que permiten transformar datos en información útil para la toma de decisiones: OSEMN (obtain, scrum, explore, model and interpret).

  1. Obtener

    2. Se recopilan datos de diversas fuentes, pueden ser preexistentes o recién adquiridos.

  2. Limpiar

    3. Consiste en el proceso de normalizar los datos según un formato predeterminado, gestionar los datos faltantes, corrección de errores en los datos y eliminación de datos atípicos para asegurar la calidad del análisis.

  3. Explorar

    4. Se trata de un análisis preliminar de los datos que se utiliza para planificar otras estrategias para su modelado. Se obtiene una comprensión inicial de los datos mediante estadísticas descriptivas y herramientas de visualización.


  1. Modelar

    2. Se utiliza software y algoritmos de machine learning para obtener información más profunda, predecir resultados y prescribir el mejor curso de acción.

  2. Interpretar

    3. Se convierte la información obtenida en datos de acción. Diagramas, gráficos y tablas son utilizados para representar tendencias y predicciones de manera clara y efectiva, con el fin de apoyar la toma de decisiones.
